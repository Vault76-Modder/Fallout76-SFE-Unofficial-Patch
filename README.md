# Fallout 76 - SFE (Save Everything) Unofficial Patch by DCHoaxer95

**Status:** Working for FO76 Version [Aktuelle Spiel-Version eintragen] (March 2026)

### What is this?
Since the original SFE (Save Everything) mod by Keretus is currently waiting for an official update on NexusMods, some mods (like Perk Loadout Manager) are broken. 
This is a temporary, unofficial `.dll` update created by **DCHoaxer95** to get SFE working again for those mods until the original author pushes the official update.

⚠️ **IMPORTANT NOTE FOR TEXT CHAT USERS:** ⚠️
This unofficial patch **cannot** make the TextChat mod work. If you are using the chat mod, this will not fix it for you. You have to wait for the official SFE release from Keretus!

### Installation
1. Go to the **Releases** section on the right side of this page.
2. Download the `dxgi.dll` (or .zip file).
3. Place it in your Fallout 76 root directory (where the Fallout76.exe is located), overwriting the old file.
